# Ask Your Business - Full Serbian COA Dictionary v1.3

This zero-dependency Railway demo expands the Serbian COA module from a starter konto preview into a fuller Kontni okvir browser for the framework: privredna društva, zadruge i preduzetnici.

## What v1.3 adds

- Full class/group/account dictionary for the Serbian standard Kontni okvir used in the demo scope.
- Class and group headings are visible but marked as non-posting rows.
- Review Center konto dropdown filters to postable account rows plus the internal REVIEW placeholder.
- Search/filter by class, type, review level, code and name.
- Export Serbian dictionary includes source metadata, account dictionary, raw-category mapping rules, posting treatments and tax treatments.

## Important

This demo is still browser-local. The account names are transcribed for the demo from the Kontni okvir attached to the rulebook for companies/cooperatives/entrepreneurs (Sl. glasnik RS br. 89/2020). The REVIEW row is an internal app placeholder and is not an official Serbian konto. A Serbian accountant must confirm final company-specific analytical accounts, posting templates and tax/PDV/SEF treatment before production use.

Test deployment with `/health`. Expected version: `1.3.0`.
